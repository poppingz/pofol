CSS 포함방법

1.inline 방법 (태그 옆에 attribute속성 입력)

  -ex) <div style="color:red"></div>

2.embed (=internal, head태그 영역에 <style></style>로 묶어서 CSS언어 입력)

  -ex)

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title></title>
    <!-- CSS -->
    <style>
      p{color:red}
    </style>
    <!--// CSS -->
</head>

3.external (링크로 파일을 불러옴,CSS구문을 HTML내에 작성하지 않고 파일을 불러옴)

 -ex) 파일 : A.CSS

      <!DOCTYPE html>
      <html lang="en">
      <head>
          <meta charset="UTF-8">
          <title></title>
          <!-- CSS -->
          <link rel="stylesheet" href="style.css"/>
          <!--// CSS -->
      </head>
